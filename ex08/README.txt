ex08 物联网安防报警器

实验功能
- 通过网页实现系统布防（Arm）与撤防（Disarm）
- 布防状态下，触摸传感器触发报警并锁定
- LED 高频闪烁报警，撤防后才停止
- 实现模拟物联网安防主机功能

 硬件接线
- LED 正极 → GPIO2 (D2)
- LED 负极 → GND
- 触摸传感器引脚 → GPIO4 (D4)

 实现逻辑
1. ESP32 建立 WiFi 连接并启动 Web 服务器
2. 网页提供布防、撤防按钮，通过 GET 请求控制状态
3. 全局变量 isArmed 判断是否布防
4. 布防时检测触摸值，低于阈值则触发报警
5. 报警后 LED 持续闪烁，直到撤防复位

使用方法
1. 修改代码中的 WiFi 名称和密码
2. 烧录程序并查看串口获取 IP 地址
3. 浏览器访问 IP 地址打开控制页面
4. 点击布防进入警戒模式，触摸 D4 触发报警
5. 点击撤防解除报警状态

 实验现象
- 未布防：触摸无反应
- 已布防：触摸 → LED 狂闪并锁定
- 撤防：LED 停止闪烁，系统恢复
