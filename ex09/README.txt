 ex09 实时传感器Web仪表盘

 实验功能
- ESP32 实时采集触摸传感器模拟量数值
- Web服务器提供数据接口，前端AJAX定时获取
- 网页实时刷新显示传感器数值，形成仪表盘效果
- 实现物联网数据采集与远程实时监控

 硬件接线
- 触摸传感器引脚：GPIO4 (D4)

 实现逻辑
1. ESP32 连接WiFi，启动Web服务器
2. 根路径返回网页仪表盘界面
3. /data 接口返回当前触摸传感器模拟值
4. 网页使用setInterval定时拉取数据并动态更新
5. 手指靠近→数值下降；离开→数值回升

 使用方法
1. 修改WiFi名称与密码
2. 烧录程序，查看串口获取IP地址
3. 浏览器访问IP，查看实时数值
4. 用手触摸/靠近D4观察数值变化

实验现象
网页中央数字实时动态刷新，手靠近时数值明显减小，离开后恢复。
